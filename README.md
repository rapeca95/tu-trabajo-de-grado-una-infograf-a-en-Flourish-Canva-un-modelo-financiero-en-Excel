# Infografía Spotify - Visualización de Datos
## MBA CESA | Materia: Visualización de Datos

---

## Tema
**"Spotify: El Imperio del Audio Digital - Análisis de datos 2015-2024"**

## Narrativa central
> Spotify pasó de ser un reproductor de música a convertirse en la plataforma de audio más grande del mundo. Pero, ¿es rentable para todos? ¿Y qué tan sostenible es su modelo?

---

## Estructura del repositorio

```
/
```

## Flujo de trabajo

```
1. Descargar CSV de /data
     ↓
2. Importar en flourish.studio
   (seguir guia_flourish_paso_a_paso.md)
     ↓
3. Exportar gráficas como PNG
     ↓
4. Abrir Canva con plantilla 800x2400px
   (seguir guia_canva_infografia.md)
     ↓
5. Insertar gráficas + diseñar secciones
     ↓
6. Exportar PDF final
```
