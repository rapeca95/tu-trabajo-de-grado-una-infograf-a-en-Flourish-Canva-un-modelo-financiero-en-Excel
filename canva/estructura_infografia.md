## Elementos Decorativos Sugeridos (Canva Library)

| Elemento | Uso | Búsqueda en Canva |
|----------|-----|-------------------|
| Notas musicales | Secciones de canciones | "music note" |
| Auriculares | Header o separadores | "headphones" |
| Ondas de sonido | Fondo o decoración | "sound wave" |
| Estrellas/Destellos | KPIs destacados | "sparkle gold" |
| Micrófono | Portada | "microphone" |
| Disco de vinilo | Sección discografía | "vinyl record" |
| Trofeo/Medalla | Récords y logros | "trophy" |
| Banderas UK | Origen del grupo | "uk flag" |
| Flecha hacia arriba | Crecimiento | "arrow up" |

## Tipografía Final Recomendada

```
Fuente 1: Montserrat
  - ExtraBold: Título portada, nombres álbumes
  - Bold: Títulos de sección, KPIs
  - Regular: Texto descriptivo

Fuente 2: Raleway  
  - ExtraBold: Números grandes de KPI
  - Light: Subtítulos, fuente de datos

Fuente 3: Space Mono (opcional)
  - Regular: Datos numéricos, fechas
  - Estética: tecnológica/streaming
```

## Guía de Espaciado

- Margen exterior: 60px en todos los lados
- Separación entre secciones: línea dorada `#FFD700` de 2px
- Padding interno de tarjetas: 24px
- Separación entre elementos: múltiplos de 16px
