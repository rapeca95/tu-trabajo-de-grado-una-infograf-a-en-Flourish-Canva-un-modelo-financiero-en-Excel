## Cómo Aplicar en Canva

1. En Canva: ir a **Colores de marca** → `Añadir color` → pegar el código Hex
2. Guardar todos los colores como paleta de marca antes de diseñar
3. Usar **Dorado #FFD700** solo para elementos de alto impacto (KPIs, títulos de sección)
4. El **fondo #0A0E27** debe cubrir toda la infografía para coherencia visual
