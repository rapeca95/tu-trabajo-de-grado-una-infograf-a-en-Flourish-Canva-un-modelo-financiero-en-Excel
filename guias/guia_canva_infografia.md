# Guía Canva: Infografía Spotify para MBA CESA

## Configuración inicial en Canva

### Formato recomendado
- **Tipo:** Infografía vertical
- **Dimensiones:** 800 x 2400 px (o A3 vertical para imprimir)
- Ir a Canva → Crear diseño → Dimensiones personalizadas → 800 x 2400

---

## Paleta de colores oficial Spotify

| Color | Código HEX | Uso |
|-------|-----------|-----|
| Verde Spotify | `#1DB954` | Títulos, acentos, datos clave |
| Negro profundo | `#191414` | Fondo principal, texto |
| Blanco | `#FFFFFF` | Texto sobre fondo oscuro |
| Gris oscuro | `#535353` | Texto secundario, bordes |
| Gris claro | `#B3B3B3` | Subtítulos, etiquetas |

---

## Tipografía
- **Títulos:** Circular Std Bold (o Montserrat Bold como alternativa en Canva)
- **Subtítulos:** Montserrat SemiBold
- **Cuerpo:** Montserrat Regular o Open Sans

## Elementos visuales recomendados en Canva

### Para descargar/buscar en Canva:
- Buscar: "spotify waveform" → ondas de audio para fondos
- Buscar: "music headphones" → iconos para secciones
- Buscar: "streaming dark" → elementos de diseño
- Usar la librería de **iconos planos** de Canva (sección Elements)

### Efectos de diseño:
- Fondo principal: negro `#191414`
- Líneas divisoras: verde `#1DB954` con 1-2px
- Sombras en cajas: sombra oscura para profundidad
- Gradientes: negro a verde oscuro en encabezado
