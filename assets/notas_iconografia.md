# Notas de Iconografía y Recursos Visuales

## Recursos Gratuitos para el Proyecto

### Imágenes del Grupo
> **Importante:** Usar imágenes con licencia libre o de dominio público.
> Para uso académico, documentar la fuente.

**Fuentes recomendadas:**
- Wikimedia Commons: buscar `One Direction band`
- Getty Images (con marca de agua para maquetado inicial)
- Unsplash: buscar `concert crowd`, `music streaming`, `british pop`

### Íconos de Plataformas (uso en Canva)
- **Spotify:** Canva library → buscar "Spotify icon"
- **YouTube:** Canva library → buscar "YouTube logo"
- **Apple Music:** Canva library → buscar "Apple Music"
- Alternativa: [Simple Icons](https://simpleicons.org/) — SVGs gratuitos de marcas

### Íconos Genéricos Musicales
- Canva Elements → buscar "music note flat"
- Canva Elements → buscar "vinyl record outline"
- Canva Elements → buscar "microphone minimal"
- Canva Elements → buscar "headphones line"
- Canva Elements → buscar "sound wave graphic"

## Datos y Fuentes de los Datasets

| Dato | Fuente Referencial |
|------|-------------------|
| Streams Spotify | Spotify for Artists / Chartmasters.org |
| Streams YouTube | YouTube Analytics / Social Blade |
| Ventas de álbumes | IFPI Global Music Report / Official Charts Co. |
| Streams Apple Music | Apple Music Charts históricos |
| Carreras solistas | Spotify Charts + Billboard |
| Hitos de carrera | Wikipedia (One Direction discography) |

> **Nota académica:** Los datos en los CSVs son estimaciones basadas en fuentes públicas disponibles. Para un trabajo académico formal, contrastar con fuentes primarias como Spotify for Artists o informes oficiales de la IFPI.
